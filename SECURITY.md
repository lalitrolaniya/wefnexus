# Security Policy

## Supported Versions

| Version | Supported |
|---------|-----------|
| 1.0.x   | ✅ Yes    |

## Reporting a Vulnerability

If you discover a security vulnerability in wefnexus, please report it
by opening a GitHub issue or contacting the maintainer directly via
the email listed in the DESCRIPTION file.

We will acknowledge the report within 48 hours and work to resolve
the issue promptly.
